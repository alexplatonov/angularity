#Angularity

A scaffolding for building modular applications using angular.js

## Prerequisites
```shell
    brew install npm
    npm install -g gulp
```
## Running locally

#### Install node modules
```shell
    npm install
```
#### Run gulp
```shell
    gulp serve
```
